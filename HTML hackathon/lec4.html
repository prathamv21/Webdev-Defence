Lecture 4: Links & Images in HTML

Watch Lecture Video

Links (Hyperlinks)

  • <a> — Anchor tag for hyperlinks
  • Use href attribute for destination URL

Images

  • <img> — Embeds images
  • Use src for filename and alt for description

Examples

<a href="https://example.com">Visit Example</a>
<img src="images/pic.jpg" alt="Sample Image">

Image Links

<a href="page.html">
  <img src="images/nav.png" alt="Navigate">
</a>
